# 💡BLINKIT SALES DASHBOARD ANALYSIS

### 📊Created an interactive Blinkit Sales Dashboard in Microsoft Excel.The main objective of this project was to transform raw grocery sales data into an easy-to-understand and visually appealing dashboard that helps in better decision-making and business analysis.

* Used **Pivot Tables**,**Pivot Charts**, **Slicers**, **KPI Cards**, and **Conditional** **Formatting** to build a dynamic and user-friendly dashboard.
* 📈 Analyzed key metrics including:

	💰 Total Sales: $1.20M

 	📊 Average Sales: $141
  
 	📦 Number of Items: 8523
  
  ⭐ Average Rating: 4.0

📉 Created visual reports for:

  * Fat Content Analysis
  * Fat By Outlet
  * Item Type Performance
  * Outlet Size Distribution
  * Outlet Location Analysis
  * Outlet Type Comparison
  * Outlet Establishment
    
🎛️ Implemented interactive slicers for filtering data by:

* Outlet Size
* Outlet Location
* Item Type

🔍 Identified key insights such as:

* Tier 3 outlets contributed the highest sales.
* Regular fat products generated more sales than low-fat products.
* Fruits & Vegetables and Snack Foods were top-performing categories.
* Medium and High-sized outlets contributed more sales than Small outlets.
* The dashboard maintained an average customer rating of 4.0, indicating good customer satisfaction.
